(2023-07-31) Integrating Tiptap text editor to Laravel with AlpineJS

When I initially searched for 'rich text editor' options, I came across several alternatives. After conducting more research, the concept of Tiptap resonated with me. Despite reading through tiptap's documentation, I encountered challenges while trying to incorporate it into my Laravel project. The idea of directly connecting tiptap to the Livewire model didn't appeal to me. Eventually, I decided to attempt integration by utilizing Alpine.js. I employed Alpine.js to temporarily store tiptap data, which could then be transmitted to Laravel via the hidden form.

Following numerous days of grappling with the integration of tiptap's JavaScript code into Alpine.js and the Vite build tools, I finally succeeded in creating a rich text editor within my Laravel project using tiptap
